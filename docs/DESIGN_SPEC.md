# UI设计规范

## 配色方案

### 主色调 - 淡蓝色系
| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#F0F7FF` | 极浅蓝，主背景 |
| 卡片背景 | `#FFFFFF` | 白色，内容区 |
| 主色 | `#5B9BD5` | 中蓝，按钮/选中 |
| 主色悬停 | `#4A8AC4` | 深蓝，交互反馈 |
| 浅色强调 | `#D6E8FA` | 淡蓝，边框/分隔 |
| 边框色 | `#B8D4F0` | 浅蓝灰，输入框边框 |

### 辅助色
| 用途 | 色值 | 说明 |
|------|------|------|
| 文字主色 | `#333333` | 深灰，正文 |
| 文字次要 | `#666666` | 中灰，说明文字 |
| 成功 | `#52C41A` | 绿色，完成状态 |
| 警告 | `#FAAD14` | 橙色，警告/提示 |
| 错误 | `#FF4D4F` | 红色，错误状态 |
| 强调点缀 | `#FF8C42` | 暖橙，重要按钮/高亮 |

## 字体规范

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 24px | 600 | 页面标题 |
| H2 | 20px | 600 | 区块标题 |
| H3 | 16px | 500 | 卡片标题 |
| Body | 14px | 400 | 正文 |
| Caption | 12px | 400 | 辅助说明 |
| Small | 11px | 400 | 标签/徽标 |

## 间距规范

- 基础单位：4px
- 组件内边距：16px / 24px
- 卡片间距：16px
- 页面边距：32px
- 区块间距：24px

## 圆角与阴影

- 卡片圆角：12px
- 按钮圆角：8px
- 输入框圆角：6px
- 卡片阴影：`0 2px 8px rgba(91, 155, 213, 0.1)`
- 悬停阴影：`0 4px 16px rgba(91, 155, 213, 0.18)`

## 布局规范

### 整体布局
```
┌──────────────────────────────────────┐  ← 窗口宽度 1200px
│  Header (60px)                        │  标题 + 步骤指示器
├──────┬───────────────────────────────┤
│ 侧边 │   主内容区                      │
│ 导航 │   padding: 32px                │
│ 200px│   最大宽度: 880px              │
│      │   居中显示                     │
├──────┴───────────────────────────────┤
│  Footer (40px)                        │  状态信息
└──────────────────────────────────────┘
```

### 创作步骤页布局
- 顶部 Stepper 显示5步进度
- 内容区居中，最大宽度 880px
- 卡片式承载每步内容
- 底部固定操作按钮（上一步 / 下一步）

### 视频库页布局
- 网格布局展示视频卡片
- 每行3-4个卡片
- 顶部搜索栏 + 筛选条件

## 组件状态

每个交互组件需覆盖以下状态：
- **默认** (default)
- **悬停** (hover)
- **激活** (active/focus)
- **加载中** (loading)
- **禁用** (disabled)
- **错误** (error)
- **空状态** (empty) — 无数据时的占位提示

## 动画

- 过渡动画：`transition: all 0.2s ease`
- 页面切换：淡入淡出 0.3s
- 卡片悬停：上浮 4px + 阴影加深
- Loading：使用骨架屏或 spinner
